# 💰 Expense Tracker App

## 🚀 Overview
A personal finance management system that allows users to track income and expenses, set budgets, and analyze spending patterns.

---

## 🛠 Tech Stack
- PHP
- MySQL
- JavaScript
- Chart.js
- HTML/CSS

---

## ✨ Features
- 💵 Income & expense tracking
- 📊 Monthly budget system
- 📈 Interactive charts (Chart.js)
- 🔍 Category-based filtering
- 📅 Date-wise analysis
