# Expense Tracker App

A personal finance management web application built with PHP and MySQL.

## Features
- User Authentication
- Add Income & Expenses
- Monthly Budget Tracking
- Category-wise Expense Management
- Charts and Reports
- Responsive Design

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Chart.js

## Installation
1. Clone the repository
2. Import the SQL database
3. Configure db.php
4. Run using XAMPP
